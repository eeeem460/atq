# atq
競技プログラミングの復習を管理するコマンドラインツール．
キュー構造を用いて復讐すべき問題を管理する．


### キューの中身を見る
```
atq look
```


### キューに問題を追加する
atq push を実行した後に，YYMMDD の形で復習する日付を入力し，その後に問題のURLを入力する．
例えば，
```
atq push
220617 
https://atcoder.jp/contests/abc253/tasks/abc253_f
```

### キューの問題を削除する
atq pop （問題番号）の形で実行する．例えば，
```
atq pop 2
```
